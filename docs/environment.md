# Environment Variables

No environment variables are currently required.

Before production integration, add `.env.example` with at least:

- `DATABASE_URL`
- `SESSION_SECRET`
- `APP_URL`
- `EMAIL_FROM`
- `EMAIL_SERVER`
- `FILE_STORAGE_BUCKET`
- `AI_PROVIDER_API_KEY`
- `AUDIT_LOG_RETENTION_DAYS`

Never commit real secrets.
